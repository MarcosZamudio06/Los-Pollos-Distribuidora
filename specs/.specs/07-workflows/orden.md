## Política de uso

- Usar este archivo solo para revisar secuencia y dependencias inmediatas.
- No usar este archivo como spec de negocio, API, base de datos, UI o testing.
- Si este archivo contradice `task/action.md`, reportar contradicción.
- Si este archivo contradice evidencia de código o validación, reportar contradicción.
- Si se necesita detalle de una TASK no activa, consultar el documento de workflows correspondiente solo si el usuario lo pide.
