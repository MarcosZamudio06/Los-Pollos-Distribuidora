# Stack Tecnológico

## Frontend

| Tecnología | Uso |
|---|---|
| React | Interfaz de usuario |
| Vite | Bundler y entorno de desarrollo |
| TypeScript | Tipado estático |
| React Router | Navegación |
| TanStack Query | Manejo de estado remoto/API |
| Tailwind CSS | Estilos |
| Axios | Cliente HTTP |

## Backend

| Tecnología | Uso |
|---|---|
| NestJS | Framework backend |
| TypeScript | Tipado estático |
| Prisma | ORM |
| PostgreSQL | Base de datos |
| JWT | Autenticación |
| Bcrypt o Argon2 | Hash de contraseñas |
| Class Validator | Validación DTO |
| Swagger | Documentación API |

## Infraestructura

| Tecnología | Uso |
|---|---|
| Docker | Contenedores |
| Docker Compose | Orquestación local |
| Nginx | Proxy reverso |
| PostgreSQL Volume | Persistencia de datos |

## Herramientas de calidad

| Herramienta | Uso |
|---|---|
| ESLint | Linting |
| Prettier | Formato |
| Jest | Pruebas backend |
| Vitest | Pruebas frontend |
| Playwright | Pruebas E2E opcionales |
